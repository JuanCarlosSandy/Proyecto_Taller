Instalas el BD y para poder activar tienen que ir a tu buscador y seguir esto pasos:
1. Editar variables de entorno
2. Variables de entorno
3. Click en Path y le das a editar
4. Nuevo y colocar la direccion en donde esta tu instalada MongoDB por ejemplo:
   C:\Program Files\MongoDB\Server\6.0\bin
5. Acepatr y Aceptar

Abres PowerShell y escribes mongod y se abre el servidor de MongoDB

Tambien necesitas instalar PYMONGO, eso lo haces en tu consola cmd:
Windows: pip install pymongo
